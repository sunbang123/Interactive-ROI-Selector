### WPF (C#)

WPF는 벡터 기반 렌더링을 사용하여 확대/축소가 매우 부드럽고, 이미지 위에 그래픽을 덧그리는 작업에 특화되어 있습니다. 나중에 만드실 '웨이퍼 시뮬레이터'의 메인 UI로 가장 강력한 후보입니다.

- **프로젝트 아이디어: 인터랙티브 ROI (관심 영역) 셀렉터**
- **구현 목표:**
    - 고해상도 사진을 불러옵니다.
    - **마우스 휠**을 굴리면 마우스 커서 위치를 중심으로 사진이 부드럽게 확대/축소(Zoom) 되게 만듭니다.
    - **마우스 드래그**로 사진 위를 이리저리 이동(Pan)합니다.
    - 사진 위에서 마우스 왼쪽 버튼을 드래그하면, 특정 영역(예: 부모님 얼굴)에 빨간색 사각형(ROI, Region of Interest)이 그려지도록 합니다.
- **비전(Vision) 학습 포인트:** 비전 소프트웨어의 핵심인 **사용자 지정 검사 영역(ROI) 설정 기능**과, 미세한 결함을 확인하기 위한 **카메라 뷰어 조작(Zoom/Pan) 기술**을 학습하게 됩니다.
<hr>

#### 추가 구현

- Ctrl + 우클릭으로 관심 구역 Zoom
- Crtl + Z 단축키로 ROI 상자 제거
